# Random Glucose-level Categories
# Normal < 140 mg/dl
# Prediabetic 140-199 mg/dl
# Diabetic > 200 mg/dl

# REMEMBER: Random Glucose test is not sufficient to decide the status, once it reaches over 200 (Require other tests as well)
